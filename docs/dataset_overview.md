# Dataset Overview

This document describes the raw CSV files used in the project.

## Raw Files

| File | Description |
|---|---|
| olist_customers_dataset.csv | Customer information and customer location data. |
| olist_geolocation_dataset.csv | Brazilian zip code prefix geolocation data. |
| olist_orders_dataset.csv | Order-level data including purchase and delivery timestamps. |
| olist_order_items_dataset.csv | Order item details including product, seller, price, and freight value. |
| olist_order_payments_dataset.csv | Payment information for each order. |
| olist_order_reviews_dataset.csv | Customer review scores and review timestamps. |
| olist_products_dataset.csv | Product information such as category and dimensions. |
| olist_sellers_dataset.csv | Seller information and seller location data. |
| product_category_name_translation.csv | English translations for product category names. |

## Notes

Raw CSV files are stored locally in `data/raw/` and are not committed to Git.
